How it all works

Got some extra space in your car?

Find the event you are driving to on the Events page or the homepage will always show the next 5 events.
Press the and create a new post with details about when and where you are leaving from.
Sit back and relax, you will recieve an email notification when someone responds to your post.
Opt - Be proactive, browse the event for 'I need a ride' posts and message them to help them out.

Need a lift?

Find the event you are going to on the Events page or the homepage will always show the next 5 events.
Press the and create a new post with details about needing a ride.
Browse the event for posts that show they have space.
If you find a post simply click on it for further details and message the user with the which will take you to a write a message page.

Can't find an event?

You can create a new event easily with the create new event form found here.
